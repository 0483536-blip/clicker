# Epic Clicker Ultimate

A super-addictive, fully-featured idle/clicker game built with pure HTML, CSS, and JavaScript – no frameworks, no build tools, just one file!

## Features
- Manual clicking with escalating multipliers  
- 4 powerful upgrades (costs increase with each purchase)  
- Auto-clickers that generate clicks per second (CPS)  
- Mega multipliers (x2 → x10 → …)  
- Auto-Buyer – automatically purchases the best available upgrade  
- Clean, responsive dark-theme UI  
- Works offline – perfect for a quick break or long idle session  

## How to Play
1. Open index.html in any modern browser  
2. Click the giant golden button to earn points  
3. Spend points on upgrades to grow faster  
4. Watch your CPS explode and let the Auto-Buyer take over  

Goal: Reach infinite clicks (or at least a ridiculously huge number)!

## Files
- index.html – the complete game (copy-paste and play instantly)

## No Installation Required
Just download or copy the single HTML file and double-click to start playing. Share it with friends – it works everywhere!
